# 账芽

![账芽图标](public/site-icon.png)

账芽 · 慢慢记，慢慢长

账芽是一个轻量、温和的记账应用，帮助你用低负担的方式持续记录日常收支，在长期积累中看见自己的消费习惯与财务变化。

## 技术栈

- Next.js
- Convex
- Better Auth
- shadcn/ui
- Bun
- Tailwind CSS v4

## 本地开发

```bash
bun install
bun dev
```

默认开发地址：`http://localhost:3000`

## 常用命令

```bash
bun dev      # 启动开发环境
bun build    # 构建生产版本
bun start    # 启动生产服务
bun lint     # 运行 ESLint
```

## 项目愿景

- 用更自然的交互降低记账门槛
- 让账目记录与复盘都足够轻盈
- 通过长期数据沉淀，支持更好的财务决策
